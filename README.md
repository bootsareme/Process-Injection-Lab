# Process Injection Lab
Open the `.sln` in Visual Studio 2022+ and build the project in *Release* mode. The executables are outputted to `x64/Release`.

## Projects

- **Victim:** A simple Blackjack program that you can play or hack. It is "vibe coded" in the sense that it is generated by GPT-5 completely, making it not the safest program out there...
- **Memory Injector:** This program asks you to scan a range of addresses in the target program's address space. Upon finding a value that matches, you can overwrite the variable to a value of your choice, thereby altering the state of the program externally.
- **DLL Injector:** This program takes a payload in the form of a dynamic link library (a module loaded at runtime). Run the program to load any DLL payload of your choice.
- **DLL Payload:** This sample DLL showcases how a program can be "hijacked" after being injected. You can modify this program and supply custom shellcode.
